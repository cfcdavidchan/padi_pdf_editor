# Auto fill-in for PADI PDF


##### Requirement:
* python >= 3.6
* Ubuntu or Mac OS
* Install Necessary Library
    ```
    pip install -r requirements.txt
    ```
* Create the coach information as a txt file inside ```./source```

  Example:
  ```
  coach Name
  PADI Code
  Dive Shop Code
  ```
* Create the coach signature as a png inside ```./source```
